## EPBC

Este es un proyecto creado con Next.js, desarrollado en TypeScript consiste en una tienda, tiene pagos personalizados en PayPal, algunas herramientas que se implementaron fueron:

* mongoose
* docker
* bcryptjs
* Material ui 
* axios 
* cloudinary
* js-cookie
* jsonwebtoken
* formidable
* next-auth
* sass
* swr


## Modo de empleo
* yarn install 
* iniciar .env
* yarn dev
